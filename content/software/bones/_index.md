## About

"Bones" is a small x86 kernel I wrote using C and nasm.

## Features
 - Serial I/O
 - Text-mode VGA driver
 - PS/2 keyboard driver
 - Handwritten bootloader with ELF loader
 - Alternatively, Multiboot2 compatible (for GRUB)
 - Has sync primitives despite lacking threads :thinking:
